  • when a feature is removed/updated we usually wait a year before removing it on website
  • we keep the index in the root directory because servers are programmed to look for it there
  • id can only be used once in a document
  • selectors (element name, class, id) can be grouped with a comma e.g., h1, h2, h3, p,
  • box model=margin (right, bottom, left); border (right, bottom, left); padding (right, bottom, left); content (width & height)
  • margins effect each other and can cause problems
  • box-sizing: border-box will return any individual browser settings to default, it will also count border with content height e.g., 3cm height + 1rem will be 3 with border-box (it will eat into content)
  • margins cannot be coloured
